## 🍰 The Sugar Atelier
“Where design meets sweetness ✨”
The Sugar Atelier is a visually elegant and responsive bakery website designed using modern front-end technologies. 
This project showcases a premium dessert brand with a focus on clean UI/UX design, smooth navigation, and aesthetic presentation.

---

## 📌 Project Overview
The Sugar Atelier is a web designing project aimed at creating a beautiful and user-friendly website for a dessert/bakery brand.
The website highlights different products, brand identity, and contact details while maintaining a luxurious and minimal design.

It demonstrates how front-end technologies can be used to build visually engaging and responsive websites.

---

## 🚀 Features
🎨 Elegant and modern UI design
📱 Fully responsive (mobile, tablet, desktop)
🧁 Attractive product showcase section
🧭 Smooth and simple navigation
✨ Clean layout with proper spacing and alignment
⚡ Fast loading and lightweight design.

---

## 🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout design
JavaScript – Interactivity 
GitHub Pages – Deployment.
